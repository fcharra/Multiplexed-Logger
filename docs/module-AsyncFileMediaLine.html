(package) fileState :string

Internal state of the file. It can either be: 'waiting': File is NOT still created or properly formatted. This is an invalid state, and operations on the file should wait until it's changed. 'blank': File is ready, but no logs have been written yet. Concrete media need to know this, to properly modify the JSON string (remove the first preppended comma.) 'initiated': File is ready, and logs have already been written to it. So no especial steps need to be taken before using it.
